# 🎮 Pac-Minder (v1.36)

**Pac-Minder** egy modern, multiplayer Pac-Man feldolgozás, amely a brutális minimalizmus és a reszponzív élmény jegyében született. A játékot a **TYBI GAMES** fejlesztette.

## ✨ Főbb jellemzők
*   **Multiplayer:** Valós idejű játékélmény Socket.io alapokon.
*   **Adaptív Layout:** Tökéletes megjelenés asztali gépen és mobilon is (1.36-os frissítés).
*   **Egyedi Design:** Sárga-Mélyzöld színpaletta, pixelesített UI elemek.
*   **Custom Characters:** Egyedi SVG karakterek (Marci és Csaba) tátogó animációval.

## 🚀 Technológiai stack
*   **Frontend:** HTML5 Canvas, Tailwind CSS, Vanilla JavaScript.
*   **Backend:** Node.js, Express.
*   **Kommunikáció:** Socket.io.

## 🛠️ Telepítés és futtatás
1. Klónozd a tárolót: `git clone https://github.com/buksifules/pac-minder.git`
2. Telepítsd a függőségeket: `npm install`
3. Indítsd el a szervert: `npm start` vagy `node server.js`
4. Nyisd meg a böngészőben: `http://localhost:3000`
